# Scalar API Docs

This is an Next.js web built by Scalar API References using OpenAPI schema generated by the `@apps/fastify`.

## Unnecessary
Actually, using another framework Next.js here is really unnecessary, because I have already built the Scalar API References from the `@apps/fastify` on its homepage with `@scalar/fastify-api-reference`. You can run the `@apps/fastify` and open its homepage.

Well, I'm still using Next.js here just for a showcase of a monorepo setup.

## Why not Swagger?
Simply, I'm not fan of the Swagger because of the old user interface. Lately, I'm using this in my current company I'm currently working with.

## Getting Started

### Fastify Development Server
You have to run the `@apps/fastify` first before running the Next.js server.
Go to the `/apps/fastify`, and run
```
npm run dev
```

### Next.js Development Server
Ensure the `@apps/fastify` is running (http://localhost:4000), Then, in this directory you can run Next.js development server by execute
```
npm run dev
```

Open http://localhost:3000
Voila!
